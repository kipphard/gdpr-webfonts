# DSGVO Webfonts & externe Anfragen

Findet **externe Anfragen** auf deiner WordPress-Seite (Google Fonts, Google Analytics,
YouTube, Maps, Gravatar, reCAPTCHA, Font Awesome, CDNs …), die personenbezogene Daten an
Dritte übertragen — und hilft, sie DSGVO-konform zu entfernen oder lokal einzubinden.
Ehrlicher Scan deines gerenderten HTML, **kein Cookie-Banner-Overlay**.

## ▶ Live-Demo (ohne Installation)

**[Demo in WordPress Playground öffnen](https://playground.wordpress.net/?blueprint-url=https://raw.githubusercontent.com/kipphard/dsgvo-webfonts/main/demo/blueprint.json)**
— der Scan-Bericht mit erkannten externen Diensten, direkt im Browser.

## Installation

Neueste [Release](https://github.com/kipphard/dsgvo-webfonts/releases)-Zip laden →
WordPress-Admin → *Plugins → Installieren → Plugin hochladen → Aktivieren*. Danach unter
dem Menüpunkt **DSGVO Webfonts** den Scan starten.

## Funktionen (kostenlos)

- **Scan & Bericht:** prüft das ausgelieferte HTML auf externe Anfragen und gruppiert sie
  nach Risiko (Hoch / Mittel / Niedrig) mit konkreten Handlungsempfehlungen
- Erkennt u. a. Google Fonts, Analytics / GA4 / Tag Manager, Google Ads, YouTube, Vimeo,
  Google Maps, Gravatar, reCAPTCHA, Meta Pixel, Hotjar, Font Awesome, JS-CDNs, WP-Emojis
- **Schnell-Maßnahmen:** Google Fonts entfernen · WordPress-Emojis (s.w.org) deaktivieren ·
  Gravatar durch lokalen Platzhalter ersetzen
- Zweisprachig (Deutsch + Englisch)
